# Smart Study Planner Generator 📚

## Overview
This project helps students create an effective study plan based on subject difficulty and available preparation time.

The planner distributes study hours intelligently so difficult subjects get more attention.

## Features
- Automatic study time allocation
- Considers subject difficulty
- Simple and easy to use
- Beginner-friendly Python project

## Technologies Used
- Python
- Dictionary data structure
- Problem-solving logic

## How it Works
1. User enters subjects
2. User specifies difficulty level
3. User enters days left for exam
4. Program calculates study hours for each subject

## Example

Subjects:
Math – Hard  
English – Easy  
Python – Medium  

Days left: 5  
Hours per day: 4  

Output:
Math → 10.0 hours  
English → 3.3 hours  
Python → 6.7 hours
